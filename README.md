<h1>EGG_LANGUAGE_PYTHON</h1>

<p>
EGG is a minimal language interpreter built using Python. It is designed for basic learning and experimentation(I just wanted to know how programing languages logic are defined), focusing only on variable storage and output printing. It is not intended for production use or any perpose just buult for .
</p>

<h2>Overview</h2>

<ul>
  <li>Printing text or variables</li>
  <li>Declaring simple variables with basic types</li>
</ul>


<h2>How to run</h2>
<p>Just download all this files and keep the directory and files name as same as repo than change you terminal dir to the main folder where you have stored all this files and folders(if you don't know how to ise git) and <h3>run the egg.py</h3> with this cmd</p>
<pre>
&gt;python egg.py
</pre>

<h2>Syntax</h2>

<h3>1. Print statement</h3>

<pre>
&gt;SHOW 'content'/var
</pre>

<p>
Prints a string value (in single quotes)
</p>

<h3>2. Variable declaration</h3>

<pre>
&gt;VAR type varname = value
</pre>

<p>
Used to define variables.
</p>


<h3>3. exit from interpreter</h3>

<pre>
&gt;EXIT
</pre>

<p>
exits from the interpreter(exit from loop)
</p>
<h3>4. Show length</h3>

<pre>
&gt;LEN var
</pre>

<p>
Shows vars length if that exists
</p>
<h3>5. Show Type</h3>

<pre>
&gt;TYPE var
</pre>

<p>
Shows vars TYPE if that exists
</p>
<h2>Supported Data Types</h2>

<ul>
  <li><b>STR</b> → for text values</li>
  <li><b>NUM</b> → for numeric values</li>
  <li><b>BOOL</b> → for boolean values</li>
</ul>
